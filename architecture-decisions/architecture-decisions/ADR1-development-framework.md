# Architecture Decision Record: Development Framework

## Summary
Choose a framework to build our Tic-Tac-Toe mobile app.

## Issue
We need a framework that fits our skills, timeline, and supports Android development.

## Decision
Decided on React Native.

## Status
Decided on React Native. Open to small configuration adjustments as needed.

## Details

### Assumptions
- Our team understands basic React from class.
- We need a simple and fast solution.

### Constraints
- Short project timeline.
- Beginner-level mobile development experience.

### Positions
We considered Ionic, Cordova, Framework7, and NativeScript.  
React Native is the best fit for our team's skills and timeline.

### Argument
React Native is easy to use, well-supported, and matches our course content.

### Implications
- Faster development  
- Easy UI components  
- Requires Android setup

## Related
- Works well with Stack Navigation  
- Supports Android devices
